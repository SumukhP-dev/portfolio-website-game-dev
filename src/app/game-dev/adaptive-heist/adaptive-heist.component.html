Adaptive Heist Game

An AI-driven heist microsimulation game built in Unreal Engine 5.7. Players plan heists and manage autonomous AI crew members who execute plans with dynamic behaviors. Developed a complete C++ foundation with 20+ classes including top-down/isometric camera system, advanced AI systems with skill-based interactions (Stealth, Hacking, Combat), panic states, and guard AI with patrol, detection, investigation, and communication. Features specialized crew types (Hacker, Muscle, Thief) with unique abilities, game management state machine, and an interaction system supporting multiple types. Demonstrates skills in C++ game development, AI programming, Behavior Trees, and Blueprint architecture. C++ foundation complete (23 commits, ~3000+ lines). Ready for Blueprint implementation and level design.
